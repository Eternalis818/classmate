# 江湖风云榜 · 主题素材

此目录存放「江湖风云榜」主题的 8 个素材位 PNG 图片。

**主题色**：墨黑 `#1F1410` / 血红 `#C84032` / 银白 `#C8B89A` / 竹青 `#8FA3B0`

**字体**：思源宋体（Noto Serif SC）

**氛围建议**：水墨山水 / 剑气 / 落日 / 远山 / 玉佩 / 卷轴（但更江湖气，少宫廷感）

## 8 个素材位（文件名固定）

| 槽位 | 文件名 | 建议尺寸 | 用途 |
|------|--------|----------|------|
| `hero-bg` | `hero.png` | 1920×1080 | Hero 全屏背景图（远山/落日/刀光） |
| `profile-frame` | `bamboo-frame.png` | 900×1100 | 个人档案卡片框（竹简/玉佩） |
| `timeline-marker` | `sword-marker.png` | 64×64 | 时间线圆点/图标（剑/刀/笔） |
| `ability-bar-bg` | `innerforce-bg.png` | 1000×40 | 功力条底图 |
| `ability-bar-fill` | `innerforce-fill.png` | 1000×40 | 功力条填充纹理 |
| `envelope-closed` | `letter-sealed.png` | 800×600 | 信笺合上（蜡封） |
| `envelope-open` | `letter-unrolled.png` | 800×600 | 信笺展开 |
| `stamp` | `wulin-seal.png` | 200×200 | 武林印章（朱红/篆书） |

## 上传方式

把同名 PNG 直接替换此目录的占位文件即可，无需改代码。

## 占位说明

当前是 1×1 透明 PNG 占位（避免 404）。主题样式在没有图片时仍可正常显示（用 CSS 渐变背景和默认字体），只是没有视觉装饰。
